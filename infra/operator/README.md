# Diom Operator

A Kubernetes operator that manages Diom cluster deployments via the `DiomCluster` custom resource.

## Usage
<!-- FIXME -- we need a proper link to the Helm chart here -->
We recommend installing the Operator via the Diom Helm chart, which manages provisioning all components needed to run Diom in Kubernetes. Please see the [Diom documentation](https://diom.svix.io) for more details.
