# Coyote Operator

A Kubernetes operator that manages Coyote cluster deployments via the `CoyoteCluster` custom resource.

## Usage
<!-- FIXME -- we need a proper link to the Helm chart here -->
We recommend installing the Operator via the Coyote Helm chart, which manages provisioning all components needed to run Coyote in Kubernetes. Please see the [Coyote documentation](https://coyote.svix.io) for more details.
