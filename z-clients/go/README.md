# Installation

```sh
go get diom.svix.com/diom/go
```

## Usage
Please refer to [the documentation](https://diom.svix.com) for more usage instructions.

## Contributing

Before opening a PR be sure to format your code!

```sh
go fmt ./...
```

## Running Tests

Simply run:

```sh
go test ./...
```
