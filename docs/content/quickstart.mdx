<CodeTab>

Gradle: Add this dependency to your project's build file:

```groovy
implementation "com.svix:coyote:0.x.y"
```

Maven: Add this dependency to your project's POM:

```xml
<dependency>
  <groupId>com.svix</groupId>
  <artifactId>coyote</artifactId>
  <version>0.x.y</version>
</dependency>
```

</CodeTab>
<CodeTab>

Gradle: Add this dependency to your project's build file:

```groovy
implementation "com.svix.kotlin:coyote-kotlin:0.x.y"
```

Maven: Add this dependency to your project's POM:

```xml
<dependency>
  <groupId>com.svix.kotlin</groupId>
  <artifactId>coyote-kotlin</artifactId>
  <version>0.x.y</version>
</dependency>
```

On macOS install via <a href="https://brew.sh/">Homebrew</a>:

```sh
brew install svix/coyote/coyote-cli
```

On Windows install via <a href="https://scoop.sh/">Scoop</a>:

```sh
scoop bucket add coyote https://github.com/svix/scoop-coyote.git
scoop install coyote
```

For other platforms, such as linux, checkout the <a href="https://github.com/svix/coyote#CLI">CLI docs</a> on Github.
